golfteamplanner
===============

Golf Team Planning Module for yii2 framework. Expects you to have a tbl_user in your database, containing the "id" field... The module includes the following functionalities

* HCP - Management
* Team Tournament Event Management
 - Dates
 - Player commitments
 - Revision
